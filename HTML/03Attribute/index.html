HTML Attributes

Understanding Attributes

HTML attributes provide additional information about an element and are always specified in the opening tag. Attributes are made up of a name and a value, separated by an equal sign.

Common Attributes

Some common HTML attributes include:

  • id: Used to uniquely identify an element within a document.
  • class: Used to specify one or more class names for an element.
  • href: Specifies the URL of the linked resource for anchor (a) elements.
  • src: Specifies the URL of the image source for img elements.
