Forms and Input Elements

Form Structure

A form in HTML is used to collect user input. It is defined using the <form> element.

Input Types

HTML provides various input types to collect different types of data. Some common input types include:

  • text: For plain text input
  • password: For password input (characters are masked)
  • email: For email input (includes validation for email format)
  • number: For numeric input
  • checkbox: For checkboxes
  • radio: For radio buttons
  • submit: For form submission
  • button: For generic button

Form Submission and Handling

Forms can be submitted using the <input type="submit"> or <button type="submit"> elements. The form data is sent to the server defined in the action attribute of the <form> element.
